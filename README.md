# ÇalıPardusLab2 Pardus Eğitim Uygulamaları Kısayol Paneli

Bu proje, Pardus işletim sistemi üzerinde eğitim ortamlarında sık kullanılan uygulamalara daha hızlı erişim sağlamak amacıyla geliştirilmiş bir kısayol paneli prototipidir.

## Proje Amacı

Öğrenciler ve öğretmenler, eğitim ortamlarında sık kullanılan uygulamalara hızlı ulaşmak isteyebilir. Bu proje ile LibreOffice, tarayıcı, dosya yöneticisi, terminal ve eğitim araçları gibi uygulamalara tek menü üzerinden erişim sağlayan sade bir yardımcı panel geliştirilmesi hedeflenmektedir.

## Hedefler

- Eğitim uygulamalarına hızlı erişim sağlamak
- Yeni kullanıcıların uygulama bulma sürecini kolaylaştırmak
- Pardus masaüstü deneyimini sadeleştirmek
- Eğitim kurumlarına uygun yardımcı araç geliştirmek

## Temel Özellikler

- Eğitim uygulamaları menüsü
- Uygulama başlatma simülasyonu
- Kullanıcı dostu terminal arayüzü
- Geliştirilmeye uygun modüler yapı
- Offline çalışma

## Hedef Kullanıcılar

- Öğrenciler
- Öğretmenler
- Pardus’a yeni geçen kullanıcılar
- Eğitim kurumları

## Kullanılan Teknolojiler

- Python 3

## Çalıştırma

```bash
python3 src/egitim_uygulamalari_paneli.py
